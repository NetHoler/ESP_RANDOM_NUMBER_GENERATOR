1. ABOUT THE SOFTWARE
---------------------
The ESP Number Generator is a high-precision tool designed for Extra-Sensory 
Perception experiments, randomness testing, and professional draws. The 
primary focus is on data integrity and user immersion.

2. CONFIGURATION FLOW
---------------------
Upon startup, the program guides the user through three stages:

   [A] LANGUAGE: Selection between Portuguese (PT) and English (EN).
   [B] TIMER: Sets the countdown duration (1 to 36000 seconds).
       Ideal for concentration or meditation periods prior to the draw.
   [C] DIGITS: Sets the length of the number (1 to 9 digits).

   NOTE: If the input boxes turn RED, the entered value is invalid or 
   outside the allowed range.

3. CONTROLS AND SHORTCUTS
-------------------------
The interface is designed to be minimalist. You can use the bottom buttons 
or the following keyboard shortcuts:

   [ Esc ] ....... Toggle Full Screen / Windowed Mode
   [  I  ] ....... Invert Colors (Dark Mode / Light Mode)
   [ Mouse ] ..... In Full Screen, move the mouse to reveal the interface

4. AUDIT SYSTEM (LOGS)
----------------------
To ensure no manipulation has occurred, all results are automatically 
recorded in a file on your Desktop:
File: "esp_audit_logs.txt"

Each log entry contains:
   - TIMESTAMP: Exact date and time of the event.
   - RESULT: The number generated by the system.
   - SEED: Hardware nanosecond value (CPU) used for randomness.
   - PREPARATION: Total session time until the number was revealed.
   - CONFIG: Selected digit and timer parameters.

5. TECHNICAL FEATURES
---------------------
* ANTI-SLEEP: The program prevents the monitor from turning off during 
  the countdown.
* AUTO-SCALING: The number automatically adjusts its size to ensure 
  maximum readability at any resolution.
* PRIVACY: The software runs locally and does not require an internet 
  connection.

6. TROUBLESHOOTING
------------------
- Buttons disappeared? 
  In Full Screen mode, the interface hides after 3 seconds to prevent 
  distractions. Simply move the mouse for the controls to reappear.

- History won't open? 
  Ensure the "esp_audit_logs.txt" file on your Desktop is not open in 
  another program that might be locking the file.
